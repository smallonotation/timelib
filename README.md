# timelib
Timelib is a c library, providing time functions for date handling. 

## Usage
```c
#include "time.h"

input_date(&day, &month, &year);
#input
isLeapYear(2000);
#output: true
day_of_the_year(21, 5, 2020);
#output: 	142
```
